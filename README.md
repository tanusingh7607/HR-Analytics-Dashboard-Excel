# HR Analytics Dashboard (Excel)

## Project Overview

This project is an interactive HR Analytics Dashboard developed in Microsoft Excel.

The dashboard helps analyze workforce performance, employee satisfaction, engagement levels, training costs, and workforce demographics using dynamic visualizations and KPI cards.

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Excel Formulas

## Dataset Information

- Total Employees: 2,845
- Employee Demographics
- Performance Ratings
- Satisfaction Scores
- Engagement Scores
- Training Costs

## Dashboard Features

- Interactive Slicers
- Dynamic KPI Cards
- Employee Status Analysis
- Gender Analysis
- Business Unit Analysis
- Department Analysis
- Training Cost Analysis
- Automated Insights Section

## Key KPIs

- Total Employees
- Average Employee Rating
- Average Satisfaction Score
- Average Engagement Score
- Average Work-Life Balance
- Total Training Cost
